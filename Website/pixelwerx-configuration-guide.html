PixelWerx Video Wall Configuration Guide

Standard Display Configurations

Configuration 1: 2×3 Grid

Dimensions: 2.0m × 3.0m (6.0m² total)
Resolution: 256 × 768 pixels
Panels Required: 12 units (Batch 2 - Red coding)
Power Requirements: 1.2kW average, 4.8kW maximum
Recommended Uses: Corporate lobbies, retail displays

Configuration 2: 3×6 Grid

Dimensions: 3.0m × 3.0m (9.0m² total)
Resolution: 384 × 768 pixels
Panels Required: 18 units (Batch 3 - Orange coding)
Power Requirements: 1.8kW average, 7.2kW maximum
Recommended Uses: Event venues, large presentations

Configuration 3: 2×2 Grid

Dimensions: 1.0m × 2.0m (2.0m² total)
Resolution: 256 × 512 pixels
Panels Required: 4 units (from any batch)
Power Requirements: 0.4kW average, 1.6kW maximum
Recommended Uses: Small meeting rooms, kiosks

Configuration 4: 1×6 Linear

Dimensions: 0.5m × 6.0m (3.0m² total)
Resolution: 128 × 1536 pixels
Panels Required: 6 units (Batch 1 - Brown coding)
Power Requirements: 0.6kW average, 2.4kW maximum
Recommended Uses: Ticker displays, narrow installations

Installation Guidelines

Pre-Installation Checklist

  • ☐ Structural analysis completed
  • ☐ Power infrastructure verified
  • ☐ Network connectivity confirmed
  • ☐ Environmental conditions assessed
  • ☐ Permits and approvals obtained

Assembly Instructions

Panel Mounting Sequence

  1. Install mounting framework
  2. Connect power distribution system
  3. Mount panels in sequence (left-to-right, top-to-bottom)
  4. Connect data cables between panels
  5. Perform system calibration

Cable Management

  • Use approved cable types for outdoor installations
  • Maintain service loops at connection points
  • Secure all cables to prevent strain on connectors
  • Label all connections for maintenance access

System Integration

Control Systems

  • Supported: NovaStar, Colorlight, Linsn processors
  • Input Formats: HDMI 2.0, DisplayPort 1.4, SDI
  • Network Protocol: Ethernet TCP/IP, Art-Net DMX

Content Management

  • Real-time video processing up to 4K input
  • Static image display with fade transitions
  • Scheduled playlist management
  • Remote monitoring and control capabilities

Maintenance Schedule

Daily Operations

Weekly Maintenance

Monthly Service

Annual Service

Troubleshooting

Display Quality Problems

Symptom: Color inconsistency between panels
Solution: Perform color calibration procedure
Prevention: Regular calibration schedule

Power-Related Issues

Symptom: Intermittent panel failures
Solution: Check power supply capacity and distribution
Prevention: Proper power infrastructure sizing

Communication Errors

Symptom: Missing or corrupt display segments
Solution: Verify data cable connections and signal quality
Prevention: Use quality cables and proper termination
